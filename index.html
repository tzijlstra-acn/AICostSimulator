Where does the AI run?hosting drives both infrastructure cost and which models you can use
Regulated NFR data usually can't sit on a vendor's shared cloud, so where the AI runs changes the cost shape — and the answer to "how can a self-hosted model have a token price."Vendor cloud → per-token API pricing, no infrastructure to run. EU-sovereign → EU-resident hosting, a small premium and commitment. On-prem / private → your own GPU cluster: a large fixed infrastructure + MLOps cost, but a low marginal cost per token (you bought the hardware, you're not billed per call). Only open / self-hostable models run here. Tier-1 banks and patient data typically need this — and it's why the all-in is far higher than a pure cloud build.
